1) ติดตั้งครั้งแรก วิธี RUN back-end
1. เข้า Documents\GitHub\OOP-LibraRead
2. ลบไฟล์ venv
3. python -m venv myenv
4. myenv\Scripts\activate
5. เข้า Documents\GitHub\OOP-LibraRead\fastapi-backend
6. pip install fastapi uvicorn
7. uvicorn main:app --reload


2) มี Project อยู่แล้ว
1. เข้า Documents\GitHub\OOP-LibraRead
2. myenv\Scripts\activate
3. เข้า Documents\GitHub\OOP-LibraRead\fastapi-backend
4. uvicorn main:app --reload
